What is a QR Code And How It Works

A QR code (quick response code) is a type of two dimensional (2D) bar code that is used to provide easy access to online information through the digital camera on a smartphone or tablet. In this project we create qr code by javascript code and connect its database to blogspot.com which act as an online database for our qr code data.
